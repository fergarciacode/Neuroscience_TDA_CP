# Neuroscience_TDA_CP

This repo contains the analysis code, results and plots of the project done in collaboration with Dr. Adrián Ponce-Alvarez consisting in the Topological Data Analysis of both the connectome topology and the critical activity (in the Contact Process)  of the Mammalian MRI (MaMI) dataset. 

A paper is being prepared.
